强大功能，简约体验

Tag Mark 提供了一套完整的书签管理解决方案，让您的数字生活更加有序。

嵌套标签系统

支持多层嵌套标签结构，一个书签可关联多个标签，摆脱传统文件夹单一归属的限制。

智能书签管理

现代化界面设计，轻松浏览、编辑和管理所有书签，批量操作提高工作效率。

快捷键支持

支持键盘快捷键操作，为高效用户提供更快捷的工作方式，提升管理效率。

深色/浅色模式

支持深色和浅色主题切换，自动适应系统设置，为您提供舒适的视觉体验。

高级搜索功能

强大的搜索系统，同时检索书签和标签内容，快速找到您需要的信息。

数据备份恢复

支持数据导出和导入功能，确保您的书签和标签数据安全，随时可恢复。

多种应用场景

无论您是学生、研究人员、专业人士还是普通用户，Tag Mark 都能满足您的需求。

学术研究

创建如 #研究/人工智能/深度学习 的嵌套标签，将相关论文、工具和参考资料关联起来，形成完整知识网络。

项目管理

为每个项目创建独立标签结构，如 #项目/网站重构/设计参考，轻松切换不同项目环境。

学习进阶

创建如 #学习/编程/React/教程 的进阶标签结构，将初级到高级的学习资源有序组织，形成完整学习路径。

特色亮点

精心设计的细节，带来卓越的使用体验。

优雅界面

简约大气的设计，提供流畅的用户体验，让书签管理成为一种享受。

高效操作

智能化的交互设计，最大程度减少操作步骤，提高工作效率。

个性化定制

数据安全

快速入门

只需几个简单步骤，即可开始使用 Tag Mark 管理您的书签。

1

安装扩展

从 Chrome 网上应用店下载并安装 Tag Mark 扩展。

2

创建标签

进入主界面，在"标签管理"选项卡创建您的首个标签结构。

3

关联书签

切换到"书签管理"选项卡，为您的书签添加标签。

4

高效搜索

使用搜索功能（Ctrl+K/⌘+K）快速定位您需要的内容。
